# 问题描述

在使用dein作为我的vim插件管理器之后，经常出现一些我无法解决的问题。

其中一个就是我使用的deoplete做为我的自动补全的插件，但是这个每次只是在我更改完vim配置之后生效一次，然后退出vim重启之后就失效了。

# 解决办法

将````````` ````````````````````````if dein#load_sate()`````````` 和对应的`endif` 删除了就好了
